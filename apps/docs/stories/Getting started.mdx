### Styling tools

This installation is optionnal but you may want to customize or create components using our style guides.
This package includes [Sass](https://sass-lang.com/) mixins, a [Tailwind](https://tailwindcss.com/) preset, various assets like icons and fonts, and also the base css that are already used for our differents frameworks implementations.

<div class="install-cmd">
```bash
npm i @khyonn/dst-styling
```

```bash
pnpm i @khyonn/dst-styling
```

```bash
yarn add @khyonn/dst-styling
```

</div>
